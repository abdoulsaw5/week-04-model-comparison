# Week 4: Model Comparison
Tested 4 AI models on 5 Cybersecurity Alert text samples to evaluate their suitability
for the  Anomaly Detection Workflow component of our Comparison
## Models Tested
- HF Sentiment (distilbert-sst-2)
- HF Zero-Shot (bart-large-mnli)
- HF NER (bert-large-NER)
- Groq Llama 3 8B
## Finding
Recommended Groq Llama 3  for the Anomaly Detection Workflow because Out of all of the models tested Grok stands as the most accurate.
